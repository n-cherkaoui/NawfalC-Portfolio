Maze-Solver

Maze-Solver is a Java program that solves any maze. It uses backtracking algorithms to explore different paths and avoid dead ends. The program is inspired by the Greek myth of Theseus and the Minotaur, where Theseus used a thread to trace his steps in the labyrinth. The program uses ‘.’ to represent the thread, ‘@’ to represent the walls, and ‘X’ to represent the exit.

Checksum-Calculator

Checksum-Calculator is a Python program that checks if a file has been changed by calculating the checksum. A checksum is a unique number based on the contents of the file. If the contents of the file change, then the checksum will also change. The program supports 8, 16, or 32 bit calculations.

Vigenere-Encoder

Vigenere-Encoder is a C program that takes in a key and a message as files, and encrypts the message using the Vigenere cipher. The program removes any non-alphabetic characters from the message, pads it with X's until it is a multiple of the key's length, and then encrypts it using the Vigenere cipher algorithm. The resulting ciphertext is displayed.
